# 💊 MedBrief

Your daily AI-powered health companion.

MedBrief is a simple web-based health tracker that lets users log their medicines and health details, and receive daily personalized reminders via email.

---

## 🚀 Features

- 🧠 AI-generated daily health summaries
- 💊 Medicine tracking with timing
- 👤 Personal health profile (age, blood group, history)
- 📩 Automated email reminders
- ⚡ Fast backend using Supabase
- 🤖 Free AI powered by Groq
- ⏰ Fully automated using GitHub Actions

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Backend: Supabase (PostgreSQL)  
- AI: Groq (LLaMA models)  
- Automation: GitHub Actions  
- Email: SMTP (Gmail)

---

## 📸 How It Works

1. User enters health details via web form  
2. Data is stored in Supabase  
3. Script fetches user data daily  
4. AI generates personalized health summary  
5. Email is sent automatically
